## 🛠 Tecnologias

As seguintes tecnologias foram usadas na construção do projeto:

- Spring boot 2.5.6
- java 17
- JUnit 5
- Mockito

## 🎲 Instalação

```shell
- Clone o repositório:
cd "diretorio de sua preferência"

$ maven 

$ java 17 

$ Abrir na sua IDE de sua preferência e start o projeto
```
